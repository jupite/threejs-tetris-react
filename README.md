# 🕹️ threejs-tetris-react

## 📋 项目介绍
这是一个使用Three.js和React开发的3D俄罗斯方块游戏，具有现代化的UI和流畅的游戏体验。

## ✨ 特性
- 3D方块渲染，带有金属质感和光泽效果
- 多角度观察视角，支持鼠标拖拽旋转
- 流畅的方块下落和旋转动画
- 计分系统和最高分记录
- 响应式设计，支持移动设备

## 🎮 控制方式
- **鼠标拖拽**：旋转视角
- **W/A/S/D**：移动方块（方向会根据视角自动调整）
- **Q**：沿X轴旋转方块
- **E**：沿Y轴旋转方块
- **R**：沿Z轴旋转方块
- **空格键**：硬降落（直接落到最底部）

## 🧙🏻 开发

```sh
npm install
npm run dev
```

## 📦 构建

```sh
npm run build
```
